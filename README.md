# iWish - Wish List Website

## Описание
Веб-приложение на Django, где пользователи могут:
- публиковать желания
- добавлять фото
- делиться ссылкой
- смотреть желания других пользователей

## Функционал
- регистрация и авторизация
- создание постов с желаниями
- приватные и публичные записи
- загрузка изображений

## Технологии
- Python
- Django
- HTML/CSS
- SQLite

## Как запустить
1. Установить зависимости:
   pip install -r requirements.txt

2. Применить миграции:
   python manage.py migrate

3. Запустить сервер:
   python manage.py runserver

4. Открыть в браузере:
   http://127.0.0.1:8000/

# iWish - Wish List Web Application

A Django-based web application where users can:
- create and share wishes
- upload images
- make posts public or private
- share unique links

## Features
- User authentication
- Create and manage wishes
- Image uploads
- Public/private visibility

## Tech Stack
- Python
- Django
- SQLite
- HTML/CSS

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Apply migrations:
   python manage.py migrate

3. Run server:
   python manage.py runserver

4. Open:
   http://127.0.0.1:8000/
